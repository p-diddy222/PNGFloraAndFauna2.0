PNG Flora and Fauna 2.0
An updated web application showcasing the diverse flora and fauna of Papua New Guinea, built using custom HTML/CSS styling and the Bootstrap framework for a fully responsive user experience.

Features
Comprehensive Database: Detailed presentation of unique plant and animal species native to PNG.

Responsive Layout: Fully optimized for mobile, tablet, and desktop viewports using Bootstrap grids.

Modern UI Components: Styled with clean typography, styled navigation elements, and Bootstrap utilities.

Built With
HTML5: Structural layout of the application.

CSS3: Custom styles, overrides, and theme design.

Bootstrap: Responsive grid system and interactive UI components.
